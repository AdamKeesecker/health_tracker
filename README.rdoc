#SPY-MD
This is a health tracker app to be used in co-operation with a Fitbit account. It shows your calorie intake, steps walked, and distance travelled within the last week. Data is automatically pulled from your Fitbit account and represented here.

##Technologies Used:
* Ruby on Rails
* Javascript
* Postgres
* Fitbit API
